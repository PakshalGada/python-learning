# Snake Game AI

This is an AI-powered Snake game using Deep Q-Learning.

## Demo

![Snake Game Demo](demo.gif)

## Features

- Neural network trained using Deep Q-Learning (DQN)
- Real-time gameplay visualization
- Automatic learning and improvement over time

## How to Run
```bash
python trainSnake.py
```

## Requirements

- Python 3.x
- PyTorch
- Pygame
- NumPy
